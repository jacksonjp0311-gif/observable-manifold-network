# OMN-SA v0.8.3 — CI / Release Boundary Separation

## Status

Operational architecture repair after v0.8.2.

## Summary

v0.8.3 formalizes the distinction between GitHub Actions CI and local release validation. CI is a lightweight smoke gate. Full evidence/report/dashboard/ledger refresh is a local release checkpoint process.

## Boundary

This release improves release discipline and repository health. It does not prove empirical validation, causality, mechanism, production readiness, AI understanding, or GMN replication.
