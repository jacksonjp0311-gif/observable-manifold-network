# v0.8.0 Metric Availability and Residual Field Hardening

## Status

Evidence schema hardening release.

## Added

- Canonical metrics.validation block.
- Canonical metrics.baseline block.
- Backward-compatible metric extraction.

- Metric availability benchmark report.

## Boundary

Metric availability is not empirical validation. It only means the metric is exposed in the evidence package and can be compared across runs.
