## Purpose

OMN-SA v0.7 evolves the repository from evidence replay to evidence comparison.

v0.6 made evidence replayable.

v0.7 asks whether evidence remains stable across multiple runs.

## v0.7 Law

    No evidence package is mature until it can be compared across runs.
    No stability claim is mature until drift is measured.
    No public benchmark is mature until the dashboard is visible.

## What This Is Not

- Not empirical validation.
- Not GMN replication.
- Not proof of causality.
- Not proof of mechanism.
- Not code correctness proof.
- Not production readiness.
- Not proof of AI understanding.

## Boundary

Evidence drift comparison measures local continuity, run-to-run stability, artifact-count stability, load integrity, and claim-status discipline.

It does not prove code correctness, patch safety, empirical validation, causality, mechanism, production readiness, AI understanding, or GMN replication.
